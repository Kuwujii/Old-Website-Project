Czym jest ECMAScript?

ECMAScript jest specyfikacją jezyków skryptowych ustandaryzowaną przez Ecma International (European Association for Standardizing Information and Communication Systems) w standardzie ECMA-262 (lub ISO/IEC 16262). Poza ES-em Ecma International unormowało również kilka innych języków programowania (C++/CLI, C#, Eiffel, Dart) oraz innych systemów informatycznych takich jak system plików FAT12 i FAT16, płyty CD-ROM oraz format plików JSON. Początkowo ECMAScript powstał aby ustandaryzowac język JavaScript lecz później na jego podstawie zaczeły powstawac inne implementacje takie jak TypeScript, JScript lub ActionScript. Języki te używane są głównie do pisania skryptów na strony internetowe które będą działac po stronie klienta jednak coraz bardziej popularne staje się pisanie w nich aplikacji działających po stronie serwera za pomocą np. Node.js.

Historia ECMAScripta

ECMA-262 oraz ECMAScript zostały stworzone i opublikowane w Czerwcu 1997 roku przez Ecma International. Zostały one oparte na podstawie powstałego rok wcześniej JavaScripta. Pierwsza edycja opracowana przez Guya Steele została przyjęta przez Generalne Zgromadzenie Ecma (Ecma General Assembly) w którym na dzień dzisiejszy znajdują się reprezentanci takich firm jak Apple, Facebook, Google, IBM, Intel oraz Microsoft itp. jednak to dopiero druga edycja ECMAScripta opracowana już przez Michaela Cowlishawa została dostosowana do wymagań standardu ISO/IEC 16262 ustanowionego przez ISO (International Organization for Standardization) oraz IEC (International Electrotechnical Commission). Od tamtego czasu ES rdzeniem na podstawie którego opiera się wiele innych języków w tym JavaScript.

Czym jest JavaScript?

JavaScript jest językiem skryptowym wysokiego poziomu opartym na specyfikacji ECMAScript będąc jednocześnie jego najpopularniejszym dialektem. Uznawany jest za jedną z 3 najważniejszych technologii używanych do tworzenia stron internetowych razem z HTMLem oraz CSSem. Początkowo został on stworzony aby umożliwić tworzenie interaktywnych elementów na stronach. Miały one być wykonywane bezpośrednio na komputerze użytkownika jednak na dzień dzisiejszy popularne jest pisanie za jego pomocą skryptów działających po stronie serwera oraz aplikacjach nie związanych ze stronami internetowymi. JS został stworzony przez Netscape Communications jednak często mylony jest z językiem Java należącym do Oracle (wtedy Sun Microsystems) na którym bazowane były niektóre jego właściwości.


Historia JavaScripta

JavaScript został zaprojektowany i stworzony przez Brendana Eicha z Netscape Communications, a następnie wydany Marcem 1996 roku. Celem Netscapu było wprowadzenie nowej technologii zanim zrobi to Microsoft i JS był ich kolejną próbą (zaraz po nieudanej współpracy z Sun Microsystems nad zaimplementowaniem języka Java w ich przeglądarce Netscape Navigator, przez co JavaScript został zaprojektowany w podobny sposób). Pomysł ten nie został dobrze odebrany i uważano, że lepszym pomysłem będzie próba implementacji innych istniejących już języków takich jak Perl lub Python. Aby obronić swoją decyzje, Eich potrzebował prototypu - napisał go w 10 dni. Pomysł przeszedł pomyślnie przez wszystkie testy pod nazwami Mocha oraz LiveScript, jednak finalnie wydany został już pod nazwą JavaScript (co spowodowało spore zamieszanie).

var szerokosc = 700, wysokosc = 600;
var canvas, ctx, klawisz;

var gracz = {
    x: null,
    y: null,
    width: 20,
    height: 100,
    update: function() {
        if (klawisz[87]) {
            this.y -= 7;
        }
        if (klawisz[83]) {
            this.y += 7;
        }
        this.y = Math.max(Math.min(this.y, wysokosc-this.height), 0);
    },
    draw: function() {
        ctx.fillRect(this.x, this.y, this.width, this.height);
    }
};

var ai = {
    x: null,
    y: null,
    width: 20,
    height: 100,
    update: function() {
        var cel = pilka.y-(this.height-pilka.bok)*0.5;
        this.y += (cel-this.y)*0.1;
        this.y = Math.max(Math.min(this.y, wysokosc-this.height), 0);
    },
    draw: function() {
        ctx.fillRect(this.x, this.y, this.width, this.height);
    }
};

var pilka = {
    x: null,
    y: null,
    przyspieszenie: null,
    bok: 20,
    predkosc: 12,
    serve: function(bok) {
        var losowa = Math.random();
        this.x = bok === 1 ? gracz.x+gracz.width : ai.x-this.bok;
        this.y = (wysokosc-this.bok)*losowa;
        var kat = 0.1*Math.PI*(1 - 2*losowa);
        this.przyspieszenie = {
            x: bok*this.predkosc*Math.cos(kat),
            y: this.predkosc*Math.sin(kat)
        }
    },
    update: function() {
        this.x += this.przyspieszenie.x;
        this.y += this.przyspieszenie.y;
        if (0 > this.y || this.y+this.bok > wysokosc) {
            var offset = this.przyspieszenie.y < 0 ? 0-this.y : wysokosc-(this.y+this.bok);
            this.y += 2*offset;
            this.przyspieszenie.y *= -1;
        }
        var AABBIntersect = function(ax, ay, aw, ah, bx, by, bw, bh) {
            return ax < bx+bw && ay < by+bh && bx < ax+aw && by < ay+ah;
        };
        var pdle = this.przyspieszenie.x < 0 ? gracz : ai;
        if (AABBIntersect(pdle.x, pdle.y, pdle.width, pdle.height, this.x, this.y, this.bok, this.bok)) {
            this.x = pdle === gracz ? gracz.x+gracz.width : ai.x-this.bok;
            var n = (this.y+this.bok-pdle.y)/(pdle.height+this.bok);
            var kat = 0.25*Math.PI*(2*n-1);
            var smash = Math.abs(kat) > 0.2*Math.PI ? 1.5 : 1;
            this.przyspieszenie.x = smash*(pdle === gracz ? 1 : -1)*this.predkosc*Math.cos(kat);
            this.przyspieszenie.y = smash*this.predkosc*Math.sin(kat);
        }
        if (0 > this.x+this.bok || this.x > szerokosc) {
            this.serve(pdle === gracz ? 1 : -1);
        }
    },
    draw: function() {
        ctx.fillRect(this.x, this.y, this.bok, this.bok);
    }
};

function main() {
    canvas = document.createElement("canvas");
    canvas.width = szerokosc;
    canvas.height = wysokosc;
    ctx = canvas.getContext("2d");
    document.getElementById("PONG").appendChild(canvas);
    klawisz = {};
    document.addEventListener("keydown", function(evt) { klawisz[evt.keyCode] = true; });
    document.addEventListener("keyup", function(evt) { delete klawisz[evt.keyCode]; });
    init();
    var loop = function() {
        update();
        draw();
        window.requestAnimationFrame(loop, canvas);
    };
    window.requestAnimationFrame(loop, canvas);
}

function init() {
    gracz.x = gracz.width;
    gracz.y = (wysokosc-gracz.height)/2;
    ai.x = szerokosc-(gracz.width + ai.width);
    ai.y = (wysokosc-ai.height)/2;
    pilka.serve(1);
}

function update() {
    pilka.update();
    gracz.update();
    ai.update();
}

function draw() {
    ctx.fillRect(0, 0, szerokosc, wysokosc);
    ctx.save();
    ctx.fillStyle = "#ffffff";
    pilka.draw();
    gracz.draw();
    ai.draw();
    var w = 4;
    var x = (szerokosc-w)*0.5;
    var y = 0;
    var step = wysokosc/20;
    while (y < wysokosc) {
        ctx.fillRect(x, y+step*0.25, w, step*0.5);
        y += step;
    }
    ctx.restore();
}

main();

Czym jest TypeScript?

TypeScript jest językiem open-source stworzonym przez Microsoft oraz będącym nadzbiorem JavaScripta (co oznacza, że każdy program napisany w JavaScripcie będzie prawidłowym programem w TypeScripcie). Finalny kod TypeScripta jest transkompilowany do JavaScripta. Język ten powstał głównie z powodu potrzeby pisania programów JSa na klasach, co w tamtym czasie nie było jeszcze zawarte w standardzie ECMAScript. Dzięki takiemu podejściu TS mógł zostać zaprojektowany tak aby ułatwić tworzenie dużych programów. Pomimo tego dalej umożliwia on programowanie typowych dla JSa aplikacji na stronach internetowych zarówno po stronie użytkownika jak i serwera. TypeScript umożliwia również używanie plików nagłówkowych istniejących już bibliotek JavaScripta (pliki te działają podobnie do plików nagłówkowych w języku C++), co umożliwia używanie zewnętrznych bibliotek JSa do programowania w TSie.

Historia TypeScripta

TypeScript został zaprojektowany oraz stworzony przez Microsoft, jednak dużą rolę w tym odgrywał Anders Hejlsberg (Główny architekt języka C# oraz twórca języków Delphi i Turbo Pascal). TypeScript pierwszy raz został pokazany publicznie w Październiku 2012 roku (po 2 latach produkcji) w wersji 0.8 i bardzo szybko zdobył sobie uznanie. Premiera nie obyła się bez problemów - pomimo bycia dopracowanym językiem był on mocno krytykowany za bycie językiem ekskluzywnym dla IDE Microsoft Visual Studio co uniemożliwiło prace z tym językiem na systemach nie należących do rodziny Windows (wsparcie dla pozostałych systemów operacyjnych zostało później zaimplementowane).
